# CRUD-Employee-Management-System-RNMySql

Employee Management System 
Simple CRUD App All in one page:
 - Department
 - Employee
 - Bonus


# Requirements
  - ReactJS
  - NodeJS
  - MySQL
  
# Run
  Frontend
  ```fend
  - cd client
  - npm start
  ```

  Backend
  ```bend
  - cd server
  - node index.js
  ```
